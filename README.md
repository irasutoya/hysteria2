# Hysteria2 一键安装脚本

## 使用方法

```shell
bash <(curl -fsSL https://raw.githubusercontent.com/irasutoya/hysteria2/main/install.sh)
```

## 帮助

```shell
Usage: install.sh [options]

Options:
  -port        设置监听端口 (默认: 443)
  -password    设置访问密码 (默认: 随机生成)
  -uninstall   卸载 sing-box 服务及所有相关文件
  -help        显示帮助信息
```
